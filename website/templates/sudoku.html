Sudoku Solver


This is a simple Sudoku Solver. Simply key in the set numbers and click "Solve the puzzle". Click a square, and it increments from blank to 1 to 2 to 3 etc. Depending on the number of clicks. It goes from 9 to blank and the cycle repeats itself.

What I did: I created a simple algorithm to solve a Sudoku puzzle. I used a simple recursion algorithm. I created a 2-dimensional array to hold the values. I never really liked what I saw on other sites, so I decided to create my own. I used a simple Div and button to create the grid (81 in total). I used CSS to make it look pretty. I used JavaScript to make it work. I used Flask to make it work on the server. I have used Heroku to host it and I use Git to push it to Heroku.

Improvements I should do: If the puzzle is not solvable, it should tell you. If there is more than one solution, it should tell you. If there is only one solution, it should tell you. I should also prevent a number being selectable if it is already in the row, column or 3x3 square.
